## MsgPanel 消息中心
#### 何时使用
> 顶导消息中心
>

#### API
|   参数   |          说明           |       类型    |     默认值    |
|:--------:|:-----------------------:|:-------------:|:-------------:|
| title | 消息标题            | string         | -            |
| content | 消息内容            | string         | -            |
| createTime | 创建时间            | string         | -            |

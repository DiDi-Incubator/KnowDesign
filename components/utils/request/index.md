## 代码演示

```Typescript
  /**
   * @method request 默认为get请求
   * @param {RequestInit | string} params 
   * @param {RequestInit} config 
   * 作为get请求使用时，如有参数传递，可自行拼接到url后，也可在config中传递 params:{key:value}
   * 
   */
  /*******使用*******/
  axios.request(params[,config])
```

按钮用于开始一个即时操作。

## 何时使用
 
使用didi规范，兼容antd。

## API

通过设置 Button 的属性来产生不同的按钮样式，推荐顺序为：`type` -> `shape` -> `size` -> `loading` -> `disabled`。

按钮的属性说明如下：

| 属性 | 说明 | 类型 | 默认值 | 版本 |
| --- | --- | --- | --- | --- |
| customtype | 自定义按钮类型 | `primary-standard` \| `secondary` \| `secondary-standard` \| `no-border` | null |  |


支持原生 antd4.x button 的其他所有属性。

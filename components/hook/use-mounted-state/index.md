## 何时使用

生命周期钩子提供检查组件的挂载状态的能力。

给出一个函数，如果组件被挂载，该函数将返回true，否则返回false。


```jsx
import { useMountedState } from 'dcloud-design';


const Demo = () => {
  const isMounted = useMountedState();

  React.useEffect(() => {
    setTimeout(() => {
      if (isMounted()) {
        // ...
      } else {
        // ...
      }
    }, 1000);
  });
};

ReactDOM.render(
  <div>
    <Demo />
  </div>,
  mountNode,
);
```

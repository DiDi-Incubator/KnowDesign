## zh-CN

展示字数提示。

## en-US

Show character counting.

```jsx
import { Input } from 'dcloud-design';

const { TextArea } = Input;

const onChange = e => {
  console.log('Change:', e.target.value);
};

ReactDOM.render(<TextArea showCount maxLength={100} onChange={onChange} />, mountNode);
```
